# Galo Cinza - Automação de Infoprodutos com IA

Plataforma que automatiza a criação de e-books, gera roteiros de vídeo viral e estratégias de marketing usando OpenAI.

## Funcionalidades

- 📚 Geração automática de e-books com GPT-4
- 🎬 Criação de roteiros virais para TikTok/Reels
- 🎯 Estratégia de marketing em comunidades
- 🔗 Integração com plataformas de vendas (Kiwify/Cakto)
